## Списки

Чтобы добавить ненумерованные списки, необходимо пункты выделить *. Например, так:
* элемент 1
* элемент 2
* элемент 3

Чтобы добавить нумерованные списки, необходимо пункты пронумеровать. Например, так:

1. Первый пункт
2. Второй пункт
